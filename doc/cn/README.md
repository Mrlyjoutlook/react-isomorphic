##Overview

##Installation
Install using npm:

```
$ npm install
```

dev:

```
npm run dev
```

dist:

```
npm run dist
```

If a node-sass error occurs:

```
SASS_BINARY_SITE=https://npm.taobao.org/mirrors/node-sass/ npm install node-sass
```
